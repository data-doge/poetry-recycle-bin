
# poetry-recycle-bin

take raw text, track versions, store all removed words

## how

### install

```
git clone https://github.com/data-doge/poetry-recycle-bin
cd 
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
